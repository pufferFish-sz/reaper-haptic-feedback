# ReaperHaptics 面板 · 使用说明

> 前提:已按 SETUP.md 装好脚本和手机 app。面板从 Actions 列表运行
> `ReaperHaptics_Panel.lua` 打开(建议加到工具栏),可停靠、可拉伸。

## 一、五个按钮

| 按钮             | 作用                            | 何时用                        |
| ---------------- | ------------------------------- | ----------------------------- |
| ① 启用震动编辑   | 建立/定位 HAPTICS 轨            | 每个工程第一次                |
| ② 插入瞬态       | 在编辑光标处插一个 25ms 事件    | 也可绑 `T` 键在鼠标处插,更快  |
| ③ 启动手机服务器 | 起本地服务器,显示手机要填的 URL | 每次开机后一次,窗口保持开着   |
| ④ 试发送选中     | 只把选中的 item 发到手机        | 单独试某一下的手感,不弹窗     |
| ⑤ 导出到手机     | 导出时间选区(无选区=整轨)       | 试整段节奏;会先确认导出文件夹 |

手机端只需设置一次:app → REAPER 调试台 → 填面板显示的 URL → 开启监听。

## 二、怎么"画"震动

全部在 HAPTICS 轨的 item 上操作,和剪音频一个手感:

| 操作                       | 效果                                            |
| -------------------------- | ----------------------------------------------- |
| 按 `T`(或按钮②)            | 插入一个瞬态("哒")                              |
| 拖 item 右边缘 ≥ 150ms     | 变成持续震动("嗡"),时长 = item 长度             |
| 拉 item 音量把手           | 强度:0dB=1.0,-6dB≈0.5,-12dB≈0.25                |
| take 名/备注写 `s=0.3`     | 锐度:0 钝(闷、深)→ 1 脆(薄、哒);不写 = 跟随强度 |
| 备注写 `type=t` / `type=c` | 强制瞬态 / 持续,无视长度                        |

设计参考:瞬态贴打击点,持续贴能量段;强度跟响度包络,锐度跟音色亮度。
瞬态间隔 ≥ 20ms 可分辨,间隔越小单发力度越弱。

## 三、日常循环

```
挪 item / 改音量 → 点⑤(回车确认路径) → 手机 1 秒内自动震 → 继续改
```

单试某一下:点面板事件列表里的行(= 在工程中选中)→ 点④。

## 四、面板下方的信息

- **事件列表**:实时显示当前会导出的事件(时间/类型/强度/锐度),有时间选区时只显示选区内的
- **导出文件夹 / 手机 URL**:当前配置,一目了然
- **底部状态栏**:每次操作的结果;导出警告(音量超 0dB、瞬态过密、持续超 30 秒)详情在 REAPER 控制台(View → Console output)

## 五、常见问题

| 现象            | 解决                                              |
| --------------- | ------------------------------------------------- |
| 手机没反应      | 服务器窗口开着吗?手机监听开着吗?app 在前台吗?     |
| 想换导出文件夹  | 点⑤,在确认框里改,会记住                           |
| 面板字太小/太大 | 直接拉窗口,字体跟随缩放;缩到最小尺寸为止          |
| 点②插错位置     | 它插在编辑光标处;想插鼠标处用 `T` 键(需 SWS 扩展) |
